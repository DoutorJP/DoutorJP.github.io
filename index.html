Sobre esta página

Essa página usa por enquanto apenas html e css para funcionar. Se algum dia eu ganhar alguma experiencia com JavaScript, irei integra-lo a página, porém, seguindo o conceito de quanto menos melhor, para deixar a página mais leve, simples, e com uma interface minimalista, levando em conta que o javascript deixa páginas muito pesadas, levando muito tempo para abrirem em redes mais lentas.

Talvez, daqui a alguns meses eu coloque alguns artigos nesta página, como tutoriais, mas nada muito serio.
