Como instalar o DWM

    [Artigos] [Sobre] [GitHub]

O dwm é um gerenciador de janelas que não é muito usado, conhecido por ser complicado de customizar. Porém, é muito versátil, leve e é muito bom para pessoas que estão buscando um desempenho melhor na hora de programar ou usar o terminal.


Instalação

[ATENÇÃO, ESTE TUTORIAL APENAS FUNCIONARÁ SE VOCÊ NÃO ESTIVE USANDO UM DISPLAY MANAGER. DESABILITE O DISPLAY MANAGER DO SEU SISTEMA PARA PROSSEGUIR.]

Instalar o dwm é facil e rápido


O primeiro passo, é ir no site do DWM e baixar o arquivo dwm 6.2 na seção de Downloads.

Com o arquivo .tar.gz baixado, rode:

tar xvf dwm-6.2.tar.gz

Entre no diretório criado pelo dwm:

cd dwm-6.2/

se listarmos o diretório, teremos vários arquivos de codigo-fonte, mas você não precisa fazer isso. O próximo passo, é compilar o dwm, ou seja, transforma-lo de um monte de código para um binário executável. para isso, digite:

make

E depois termine a instalação com:

sudo make install

Digite sua senha e a instalação estará completa.

Reinicie o computador, e depois faça login em algum tty.

Feito isso, você precisará usar o seu editor de preferencia (no meu caso o neovim) para editar o arquivo de inicialização do servidor X:

nvim ~/.xinitrc

Insira o conteúdo

exec dwm &

Salve o arquivo, depois, rode startx, e o dwm deverá iniciar com sucesso.
