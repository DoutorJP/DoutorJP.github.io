#GiveUpGithub: O Movimento para abandonar-mos o Github

24-07-2022

João Pedro M.V

O GitHub é uma ferramenta que está a anos no mercado, utilizada por muitos, como a única maneira de publicar softwares livres e/ou de código aberto: Grande parte de todos os projetos de software do mundo estão hospedados lá; Porém, ultimamente, uma ferramenta muito controversa foi lançada, e ela acaba de ficar mais controversa.

O Github Copilot é uma ferramenta desenvolvida pelo github, para "facilitar" o desenvolvimento de software, e seu funcionamento o torna quase como um "Assistente de Código", pois apenas digitamos na tela o que queremos que ele faça, e ele programa; porém, o problema começa na forma que ele funciona: Sendo uma inteligencia artificial, ele precisa ser treinado de alguma forma para produzir o código desejado, e a forma encontrada de treina-lo foi escaneando códigos que já estão no próprio github, assim, ele é treinado por pessoas que trabalharam sem seu consentimento para alimentar uma inteligencia artificial, e ai vem a pior notícia: Ele se tornou pago. Ou seja, estamos escrevendo códigos, sem receber dinheiro algum, para treinar uma ferramenta que só pode ser utilizada agora via um "serviço de assinatura" de "apenas" $44 (dolar), variando o preço para até $231 no plano empresarial.

Por que Abandonar o Github?

  • Foi adquirido pela Microsoft, uma empresa não muito boa no quesito direitos dos usuários
  • Enquanto o Github diz ser pró FOSS, o próprio serviço não é de código aberto, muito menos livre.
  • Entre Outros

Quais as alternativas?

  • Codeberg
  • Sourcehut
  • Gitea
