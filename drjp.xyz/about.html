O que eu uso

Sistema Operacional

Eu uso algumas ferramentas em meu computador, tanto para uso pessoal, tanto para fins de aprendizado. Uso o Gentoo Linux como "distro" (Gentoo é na verdade uma meta-distribuição, na qual o usuário monta e customiza o sistema por completo, tendo no fim um sistema rápido para o meu hardware.) principal, mas eu também rodo o Artix Linux no meu SSD

Como editor de texto, eu uso o neovim, que é um poderoso editor de texto, que pode ser configurado como IDE. tem inumeras funções e é rapido e eficiente, porém tem uma curva de aprendizado muito longa.
